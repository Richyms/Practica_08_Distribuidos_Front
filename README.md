# Vault Art - Frontend
Frontend para un servicio de streaming seguro (Vault Art)

## Estructura del proyecto
src/

├── assets/             # Recursos estáticos (imágenes, icons)

├── components/         # Componentes reutilizables

├── hooks/              # Hooks

├── pages/              # Páginas de la aplicación

├── services/           # Servicios API y llamadas HTTP

└── styles/             # Estilos y temas CSS

## Comenzar

### Prerrequisitos

- Node.js
- pnpm

### Instalación

1. **Instalación de pnpm (PowerShell)**
   ```bash
   Invoke-WebRequest https://get.pnpm.io/install.ps1 -UseBasicParsing | Invoke-Expression
2. **Clonar el repositorio**
   ```bash
   git clone https://github.com/erick-0511/VaultArt_frontend.git
3. **Instalación de dependencias**
   ```bash
   pnpm install

### Ejecución
1. **Ejecutar comando**
   ```bash
   pnpm run dev
